You can download the raw CSV data (about 1 MiB) used for these diagrams. You can also download a JSON version with slightly more data (about 10 MiB, contains commit hash and human-readable timestamp). WARNING: Your browser will try to open and display it directly, which may be annoying.

You can tinker with this code at the Github repository. If you want to hot-start your cache, you can use the hot cache used to generate above plots (about 3 MiB). Not that it's in version 2, so including the counts of DeprecatedString, and incompatible with the old cache.
